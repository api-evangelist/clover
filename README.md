# Clover (clover)

Clover is a Fiserv-owned point-of-sale platform for small and mid-size merchants. The Clover developer platform exposes the Clover REST API for inventory/orders/customers/transactions, the Ecommerce API for online payments, an Android Payments API and Clover Android SDK for native apps on Clover devices, the REST Pay Display API for semi-integrated POS, the Remote Pay SDKs (cloud, Windows, iOS, Android), and the Clover Go SDK.

## APIs
- **Clover REST API** — merchant inventory, orders, customers, employees, payments. Docs: https://docs.clover.com/reference
- **Clover Ecommerce API** — online payments, tokenization, recurring, gift cards.
- **Clover Android SDK** — native apps on Clover devices.
- **Clover Android Payments API** — abstracted payment processing on devices.
- **Clover REST Pay Display API** — semi-integrated POS Pay Display.
- **Clover Remote Pay SDKs** — Cloud, Windows, iOS, Android.
- **Clover Go SDK** — mobile card-present payments.
